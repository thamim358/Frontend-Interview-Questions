## 📖 Overview

The **Frontend React Master Study Kit** is a comprehensive, free, open-source interview preparation resource for React Frontend Developers, Senior Frontend Engineers, and React Architects. It contains **420+ high-value interview questions** across **12 topic areas**, curated from real interview patterns at top tech companies.

Every question includes detailed answers, code examples in JavaScript/TypeScript, interview strategies, and priority ratings (P1-P3) to help you focus your study time effectively.

> 💡 **For the best experience**, open the HTML files in Chrome and press `Ctrl+P` → "Save as PDF" to create premium-looking PDF study guides with the built-in dark theme.

## 🚀 How to Use

### Quick Start

1. **Navigate to the study kit**
   ```bash
   cd FRONTEND-REACT-MASTER-STUDY-KIT
   ```

2. **Open the dashboard**
   - Open `index.html` in **Google Chrome** for the best experience
   - Browse topics, search questions, and navigate the roadmap

3. **Start studying**
   - **P1 questions first** — these appear in nearly every interview
   - **P2 next** — for breadth and confidence
   - **P3 for senior roles** — deep-dive topics for Staff/Lead positions

### Creating PDF Study Guides

This kit is designed to create beautiful, print-optimized PDFs:

1. Open any HTML file in **Google Chrome**
2. Press **`Ctrl+P`** (Windows) or **`Cmd+P`** (Mac)
3. Set destination to **"Save as PDF"**
4. Under **More settings**:
   - Paper size: **A4**
   - Margins: **None**
   - Check **"Background graphics"** (preserves the dark theme)
5. Click **Save** — your premium PDF is ready!

## 📅 Study Plan

### Suggested Learning Path

| Phase | Topic | Duration | Weight |
|-------|-------|----------|--------|
| 1 | JavaScript ES6+ | 2-3 weeks | 🟢 Foundation |
| 2 | TypeScript | 1-2 weeks | 🟢 Foundation |
| 3 | React Core | 3-4 weeks | 🔴 Core |
| 4 | React Advanced | 2-3 weeks | 🔴 Core |
| 5 | Next.js | 3-4 weeks | 🔴 Core |
| 6 | CSS & Styling | 2 weeks | 🟠 Important |
| 7 | State Management | 2 weeks | 🟠 Important |
| 8 | Testing | 2-3 weeks | 🟠 Important |
| 9 | Build Tools & DevOps | 2 weeks | 🟡 Niche |
| 10 | System Design | 2-3 weeks | 🟡 Senior |
| 11 | Behavioral + Company | 2 weeks | 🏁 Final |
| 12 | Coding Interview | 3-4 weeks | 🏁 Final |

### Study Tips

- **🎯 Start with P1**: Master all P1 (red) questions first — they cover 80% of interview scenarios
- **🔄 Progressive Learning**: JavaScript → TypeScript → React Core → React Advanced → Next.js, then branch out
- **📝 Practice Aloud**: Read the "Real Interview Answer" sections and practice saying them aloud
- **💬 Prepare STAR Stories**: Have 7-8 behavioral stories ready with quantified impact from your React projects
- **🗺️ Follow the Roadmap**: Open `Frontend-Master-Roadmap.html` for detailed phase-by-phase guidance
